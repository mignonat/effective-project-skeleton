Init the project usng git source repo :

    git clone https://github.com/mignonat/healer


Install global dependencies :

    npm install --global gulp-cli

    sudo npm install --global forever //https://www.npmjs.com/package/forever

    sudo npm install --global forever-monitor //https://www.npmjs.com/package/forever-monitor


Install app dependencies :

    npm install
    	=> download all the dependencies in package.json


Run :

    npm run build-lib
         => concat all lib in one file

    npm run build-app
        => concat in one file and transpile code in 'es-2015'

    npm run start-front-dev
        => run the app in dev mode (using nodemon)

    npm run start-front-prod
        => normal startup

Update node version :
    sudo npm cache clean -f (force) clear you npm cache
    sudo npm install -g n install "n" (this might take a while)
    sudo n stable upgrade to lastest version
